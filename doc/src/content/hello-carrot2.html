Hello, Carrot2!

This article is an overview of Carrot2 and the available resources.

Carrot2 is a programming library for clustering text. It can automatically discover groups of related documents and label them with short key terms or phrases.

The process of taking raw text and discovering related documents is called cluster analysis and there are many ways to do it. Carrot2 provides common infrastructure for a few different algorithms so that your code can switch between them at will.

Out-of-the-box, Carrot2 distribution comes with:

  • a Java programming interface (API) and an implementation of several clustering algorithms,
  • a HTTP/JSON (REST) service for mash-ups or integration with languages other than Java,
  • a simple search-engine-like demo application that clusters search results,
  • code snippets and examples for reuse in your code.

Additionally, several downstream projects provide integration between Carrot2 and popular document retrieval services:

  • Apache Solr has built-in support for clustering search results via Carrot2 algorithms,
  • elasticsearch-carrot2 plugin provides search results clustering for Elasticsearch.

Try Carrot2 now

The quickest way to try out Carrot2 is to use the public live demo. This demo lets you play with clustering of search results aggregated from many popular search engines (courtesy of Comcepta AG) or explore medical documents from the PubMed database of medical abstracts.
