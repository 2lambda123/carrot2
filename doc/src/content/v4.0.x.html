Lingo3G 2.0.x

Release history for Carrot2 4.0.x (and bugfix releases).

Version 4.1.0 [unreleased]

This is a minor release with minor changes to REST API formats.

  • GH-38: [DCS] Expose the language and algorithm of all resolved template in /list. The response format of the endpoint has changed. Previously, the templates element was a list of template names, now it will contain an object with template names as keys and template content as values, for example: 

    ...
  "templates" : {
    "template1" : {
      "language" : "English",
      "algorithm" : "algorithm1"
    },
    "template2" : {
      "algorithm" : "algorithm1"
    }
  }

  • GH-37: [build] Only recompile code under IntelliJ IDEA (improve dev experience).

Version 4.0.5 [unreleased]

This is a bugfix release.

  • GH-35: [DCS] Added request processing/ service information to clustering response. This information is optional and is returned when serviceInfo HTTP parameter is enabled on a clustering request.

  • GH-34, CARROT-1280: [DCS] Clustering of documents containing multi-value fields (array of strings) fails with an exception.

  • GH-33: [DCS] Upgrade Jackson dependency to 2.11.3.
