Language Components

In this section we describe means to tune the language component resources in the REST API (DCS).

The Java API language components section already introduced the role and importance of tuning language components properly.

In the REST API, all default dictionaries reside under the dcs/web/service/resources path, relative to DCS distribution package.

Changes to language resource files are not automatically detected or reloaded, a server restart is required for changes to take effect.

Ephemeral dictionaries

All Carrot2 algorithms provide support for ephemeral dictionaries via the Java API. These dictionaries can be used to provide per-request filtering to exclude stop words or cluster labels. This functionality is not part of the REST API as such but is a part of each algorithm's set of parameters. The parameter is called the same (dictionaries) across all algorithms by convention.
