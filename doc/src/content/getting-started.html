Getting started

This article is an overview of Carrot2 and the available resources.

Carrot2 is a text clustering engine. It was designed to automatically organize small collections of documents, such as search results, into thematic categories.

Out-of-the-box, Carrot2 comes with:

  • REST API,
  • Java API,
  • search results clustering plugin for Solr and ElasticSearch,
  • demo GUI applications.

Try Carrot2

To try Carrot2 with public search engine results, use the live demo. The demo lets you cluster web search results and references from the PubMed database of medical abstracts.
